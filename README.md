# Rakta-Vahini – Smart Blood Donor Management App

## 📌 Project Overview

Rakta-Vahini is an Android-based healthcare application developed to simplify emergency blood donor searching and management. The application helps hospitals, patients, and emergency responders quickly find eligible blood donors based on blood group, nearby location, and donation eligibility status.

The project focuses on improving emergency healthcare response through a structured digital blood donor platform instead of traditional WhatsApp message forwarding and manual donor searching methods.

---

# 🚀 Features

* Blood donor registration
* Blood group-based donor search
* Location-based donor filtering
* 90-day donor eligibility verification
* Emergency donor contact support
* Donation history tracking
* User-friendly Android UI
* Secure and organized donor management

---

# 🛠️ Technologies Used

- Kotlin & Jetpack Compose  
  Uses Kotlin programming and declarative UI toolkit for modern Android app development.

- Firebase Realtime Database  
  Handles real-time donor registration, storage, and donor search functionality.

- Google Services  
  Integrates Firebase and cloud services using google-services.json configuration.

- Material 3 & Icons  
  Implements modern Material Design UI components and vector icons.

- Native Intents  
  Connects the application with the phone dialer for emergency donor calling.

- Gradle (Kotlin DSL) & AndroidX  
  Manages project dependencies and supports modern Android development libraries.

---

# 📱 Application Workflow

1. User registers donor details
2. Donor information is stored securely
3. User searches donors using blood group
4. System applies location and eligibility filters
5. Eligible donors are displayed
6. User contacts donor during emergency
